# aznoqmous/enjmin-procedural-generation
A procedural generation assignment in godot
![final planet shot](/readme.gif)

## Presentation

The final project is composed of several procegurally generated layers as can be seen bellow : 
- The ground/elevation mesh (on the right)
- The water mesh which contains elevation information (on the left)
- A buildings (or foliages) multimesh instance
A bonus skybox shader is used to emulate stars and a surrounding galaxy
